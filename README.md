# Azure-VNet-to-VNet-VPN-Gateway-Walcz
Konfiguracja połączenia Azure VNet‑to‑VNet VPN Gateway v2 dla Starostwa Powiatowego w Wałczu. Projekt zapewnia szyfrowaną komunikację między sieciami wydziałów oraz integrację kluczowych systemów, umożliwiając bezpieczną wymianę danych i dalszą rozbudowę środowiska.
